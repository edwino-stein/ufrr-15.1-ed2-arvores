Trabalho 1 - Árvores Documentação
O objetivo do trabalho é a implementação das rotinas de manipulação das estruturas de dados árvores através de uma biblioteca e a utilização da mesma em uma aplicação que interprete um arquivo contendo instruções de inserção, remoção, busca e percurso para as árvores Binária, AVL e B.

Pré-Requisitos

A aplicação e a biblioteca foi desenvolvida inteiramente em C++ em sua revisão de 2011, utilizando o compilador da GNU G++. Para geração da biblioteca de árvores, foi utilizado a ferramenta AR pra criação de bibliotecas estáticas em sistemas baseados em Unix.

Os códigos foram testados e compilados nos ambientes:

  • Mac OS X 10.10.3 com GNU G++ 4.2.1;
  • Ubuntu 15.04 com GNU G++ 4.9.2;

Compilar e Executar

Para que o programa seja compilado e executado, basta executar o arquivo de shellscript run.sh. Este script é responsável por compilar todas as dependências e executar o programa ao final.

$ chmod +x run.sh
$ ./run.sh arquivo_1 arquivo_2 ... arquivo_n

Observações:

  • Os parâmentos passados para o script run.sh, serão os mesmo utilizados passados para o programa.
  • Caso o programa já esteja compilado, o script irá apenas executar o programa.
  • Os arquivos gerados durante a compilação ficaram no diretório build.
